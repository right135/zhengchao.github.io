科研成果

StructSparse: Structure-Aware Prompt Sparsity for Hierarchical KVCache in Agent Serving 提出面向结构化 Agent Prompt 的稀疏感知机制，将 KVCache 按语义重要性分层放置于 HBM/Host/SSD，并结合 RoPE 解耦完成复用判断与请求路由调度，降低 TTFT 与缓存 IO。拟投 ASPLOS。
SuperRAG: Document-oriented KVCache Reuse for Efficient Retrieval-Augmented Generation Serving 提出面向文档的 KVCache 复用机制，降低 RAG Serving 中重复文档上下文带来的推理开销。已投至 ATC。
RAGPULSE: A Real-World RAG Workload Capturing Authentic Temporal Reuse Patterns for Efficient Serving 构建真实 RAG 工作负载，刻画业务场景中的时间复用模式，为缓存与服务调度优化提供数据支撑。已投至 EMNLP。
